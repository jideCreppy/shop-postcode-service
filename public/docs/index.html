Stores API endpoints

Add a new store

Example request:
curl --request POST \
    "http://localhost/api/shops" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"fyvqcppenztxlkoptkax\",
    \"latitude\": \"57.084444\",
    \"longitude\": \"-2.255708\",
    \"status\": \"O\",
    \"store_type\": \"shop\",
    \"max_delivery_distance\": 11
}"

const url = new URL(
    "http://localhost/api/shops"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "fyvqcppenztxlkoptkax",
    "latitude": "57.084444",
    "longitude": "-2.255708",
    "status": "O",
    "store_type": "shop",
    "max_delivery_distance": 11
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/shops

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

name   string   

Must be at least 5 characters. Must not be greater than 100 characters. Example: fyvqcppenztxlkoptkax

latitude   string   

The postcodes latitude. Must match the regex /^[-]?(([0-8]?[0-9]).(\d+))|(90(.0+)?)$/. Example: 57.084444

longitude   string   

The postcodes longitude. Must match the regex /^[-]?((((1[0-7][0-9])|([0-9]?[0-9])).(\d+))|180(.0+)?)$/. Example: -2.255708

status   string   

The status. C for closed and O for open. Example: O

Must be one of:
  • C
  • O
store_type   string   

Example: shop

Must be one of:
  • takeaway
  • shop
  • restaurant
max_delivery_distance   integer   

Must be at least 0. Must not be greater than 100. Example: 11

    \"distance\": 18
}"

Request      

POST api/shops/search

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

postcode   string   

Must match the regex /^([Gg][Ii][Rr] 0[Aa]{2})|((([A-Za-z][0-9]{1,2})|(([A-Za-z][A-Ha-hJ-Yj-y][0-9]{1,2})|(([A-Za-z][0-9][A-Za-z])|([A-Za-z][A-Ha-hJ-Yj-y][0-9][A-Za-z]?))))\s?[0-9][A-Za-z]{2})$/. Example: GIr 0AA|Q04|xg94|v4T|cq4j)))s1jW)

distance   number  optional  

Must not be greater than 100. Example: 18
